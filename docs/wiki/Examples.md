## Agent-Specific Usage

### Antigravity (Gemini)

Uses workflows with `/` prefix:
```
/debug "error message"
/plan-feature "description"
/audit security
```

### Claude Code

Uses `@/` prefix:
```
@/debug "error message"
@/plan "description"
@/audit security
```

### Cursor

Skills and rules active automatically via:
- `.cursor/rules/*.mdc` files
- Context-aware activation
