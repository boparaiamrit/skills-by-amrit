# /fix-issue Command Examples

> Diagnose and fix GitHub issues.

## Usage

```
@/fix-issue [issue-number-or-description]
```

## Examples

```
@/fix-issue #234
@/fix-issue "Login button not working on Safari"
```

## Process

1. Read issue description
2. Reproduce the problem
3. Investigate root cause
4. Apply fix
5. Add regression test
6. Link PR to issue
