# /deep-audit Command Examples

> Brutal 5-pass exhaustive audit.

## Usage


1. **Build** — Clean build verification
2. **Routes** — Every route documented
3. **Data Flow** — All API calls traced
4. **User Flows** — Critical paths tested
5. **Edge Cases** — Error states covered

Comprehensive report with 47+ checks and task generation.
